## 项目简介
html_dhym 是使用纯 HTML 开发的个人导航页面，仅单文件 index.html，无 JS、CSS 外部依赖，部署简单，本地双击即可打开。

## 文件目录
```
html_dhym/
├── index.html    # 导航主页源码
├── 预览.png      # 页面效果图
├── LICENSE       # MIT开源协议
└── README.md     # 项目说明文档
```

## 快速使用
### 方式1：本地直接打开
1. 下载项目源码并解压
2. 双击 `index.html` 即可在浏览器运行导航页

### 方式2：部署到服务器
将全部文件上传至静态网站服务器、Gitee Pages、GitHub Pages、Vercel 等平台即可在线访问。

## 项目特点
- 100% HTML 编写，无第三方库依赖
- 体积小巧，加载速度快
- 可自由修改链接、配色、布局
- 开源免费，MIT协议可商用/二次修改
